LIRI BOT

LIRI stands for Language Interpretation and Recognition Interface. LIRI-bot is a command line Node based app that takes in certain input and returns data. In this project, LIRI bot will prompt you to enter either one of the following possible actions:

  • Search on Spotify for an input song.
  • Search on OMDB for an input movie.
  • Search on Bands in Town for concerts of an input band or single artist.
  • Search on any of the previous API's based on a text file that will point LIRI to an action and an input search key.


In order to beautify the input, this app uses the "inquirer" NPM package. This limits the actions asked to Liri-Bot to the ones it has and none others. Also, it guides the user to type in the second argument needed for the calls to be successful.



You can take a look at the project repo at: LIRI-BOT_repo.
